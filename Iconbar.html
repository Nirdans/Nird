ICON BAR

An Icon Bar provides a menu to quickly navigate a Website. Icon Bar can be horizontal or vertical, with the labels below the icons or to the right.

In the example below " fa fa- " is the code for the icons.

Horizontal Icon Bar 



<!DOCTYPE html>
<html>

<meta name="viewport" content="width=
device-width, initial-scale=1">

<!-- This is icon library --> 

<link rel="stylesheet" href="https://
cdnjs.cloudflare.com/ajax/libs/font-
awesome/4.6.3/css/font-awesome.min.css">

<style>
body {margin:0}

.icon-bar {
    width: 90px;
    text-align: center;
    background-color: #555;
}

.icon-bar a {
    padding: 16px;
    display: block;
    transition: all 0.3s ease;
    color: white;
    font-size: 36px;
}

.icon-bar a:hover {
    background-color: #000;
}

.active {
    background-color: #48d1cc
    !important;
}

</style>

<body>

<div class="icon-bar">

<a class="active" href="#">
<i class="fa fa-home"></i></a>

<a href="#"><i class="fa fa-search">
</i></a>

<a href="#"><i class="fa fa-photo">
</i></a>

<a href="#"><i class="fa fa-globe">
</i></a>

<a href="#"><i class="fa fa-trash">
</i></a>

</div>

</body>
</html>

Result For Above Example



Vertical Icon Bar 



<!DOCTYPE html>
<html>
<meta name="viewport" content="width=
device-width, initial-scale=1">

<link rel="stylesheet" href="https://
cdnjs.cloudflare.com/ajax/libs/font-
awesome/4.6.3/css/font-awesome.min.css">

<style>


.icon-bar {
    width: 100%;
    text-align: center;
    background-color: #555;
    overflow: auto;
}

.icon-bar a {
    width: 20%;
    padding: 12px 0;
    float: left;
    transition: all 0.3s ease;
    color: white;
    font-size: 36px;
}

.icon-bar a:hover {
    background-color: #000;
}

.active {
    background-color: #48d1cc
    !important;
}

</style>

<body>

<div class="icon-bar">
  <a class="active" href="#">
  <i class="fa fa-home"></i></a>

  <a href="#"><i class="fa fa-search">
  </i></a>

  <a href="#"><i class="fa fa-photo">
  </i></a>

  <a href="#"><i class="fa fa-globe">
  </i></a>

  <a href="#"><i class="fa fa-trash">
  </i></a>

</div>

</body>
</html>
