HTML STYLES :

In HTML every element has a default style which is (background color is white, text color is black, text-size is 12px ....) Changing the default style of an HTML element, can be done with the style attribute .

The HTML style attribute has the following syntax :

style = "property:value"


BACKGROUND COLOR :

Background color property changes the background color .

Example for Background color : 


<!DOCTYPE html>

<html>

<body style="background-color:

lightgrey">

<h1>Background color</h1>

<p>Background color changed.</p>

</body>

</html>

TEXT COLOR :

Text color property changes the text color for an HTML element .

Example for Text color : 


<!DOCTYPE html>

<html>

<body>
<h1 style="color:RoyalBlue">

This is a heading</h1>

<p style="color:SteelBlue">

This is a paragraph.</p>
</body>

</html>

Result for the above Example :

This is a heading

This is a paragraph.



TEXT FONTS :

The font-family property defines the font to be used for an HTML element .

Example for Text fonts : 


<!DOCTYPE html>

<html>

<body>

<h1 style="font-family:courier">

Heading in courier font</h1>

<p style="font-family:PT Serif">

paragraph in PT Serif font.</p>

</body>

</html>

Result for the above Example :

Heading in courier font

paragraph in PT Serif font.



TEXT SIZE :

The font-size property defines the text size to be used for an HTML element .

Example for Text size : 


<!DOCTYPE html>

<html>

<body>

<p style="font-size:180%">

font size is 180%</p>


<p style="font-size:120%">

font size is 120%</p>

</body>

</html>

Result for the above Example :

font size is 180%

font size is 120%.



TEXT ALIGNMENT :

The text-align property defines the horizontal text alignment for an HTML element

Example for Text alignment :


<!DOCTYPE html>

<html>

<body>
<h1 style="text-align:center">

Heading is Centered</h1>

<p>This is a paragraph.</p>
</body>

</html>

Result for above Example :

Heading is Centered

This is a paragraph.
