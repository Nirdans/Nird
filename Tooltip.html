TOOLTIP

The tooltip is a common graphical user interface element. It is used in conjunction with a cursor, usually a pointer. The user hovers the pointer over an item, without clicking it, and a tooltip may appear as a small "hover box" with information about the item being hovered over.

    border-width: 5px;
    border-style: solid;
    border-color: #555 transparent;

/* Show the tooltip text when you
mouse over the tooltip container */ 

.tooltip:hover .tooltiptext {
    visibility: visible;
    opacity: 1;
